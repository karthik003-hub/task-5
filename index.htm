MATHS FORMULA

a2 – b2 = (a – b)(a + b)

(a + b)2 = a2 + 2ab + b2

a2 + b2 = (a + b)2 – 2ab

(a – b)2 = a2 – 2ab + b2

(a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca

(a – b – c)2 = a2 + b2 + c2 – 2ab + 2bc – 2ca

(a + b)3 = a3 + 3a2b + 3ab2 + b3 ; (a + b)3 = a3 + b3 + 3ab(a + b)

(a – b)3 = a3 – 3a2b + 3ab2 – b3 = a3 – b3 – 3ab(a – b)

a3 – b3 = (a – b)(a2 + ab + b2)

a3 + b3 = (a + b)(a2 – ab + b2)
